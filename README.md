# API Tester Tool

API Tester Tool is a simple open source tool that allows developers to test APIs quickly.

## Features

- Send GET requests to APIs
- View JSON responses
- Lightweight and easy to use
- Useful for testing public APIs

## Example API

https://jsonplaceholder.typicode.com/posts

## Technologies

- HTML
- JavaScript
- REST APIs

## Purpose

This project was created to help developers test APIs easily in a simple interface.
